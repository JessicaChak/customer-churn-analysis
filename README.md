# Customer Churn Analysis

## Project Overview
End-to-end analysis of telcom customer churn using SQL, Python,
and BI tools. Identifies the key churn drivers and provides
retention insights.

## Tools Used
- **SQL** (SQLite): 10 business-question queries
- **Python** (pandas, matplotlib, seaborn): EDA and visualisations
- **Tableau Public**: Interactive dashboard

## Dataset
Telco Customer Churn (Kaggle/IBM) — 7,043 customers, 21 features

## Key Findings
1. Month-to-month contracts churn at **42.7%** vs 2.8% for two-year
2. Electronic check payment correlates with **45.3%** churn rate
3. Customers with tenure under 12 months are most at risk
4. Fiber optic users churn significantly more than DSL users
